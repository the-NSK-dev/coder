# JAVASCRIPT STANDARD

## USE
- const and let for variable declarations
- strict equality comparisons
- async and await over raw promise chains
- template literals for string composition
- arrow functions for callbacks

## DO_NOT
- var declarations
- loose equality comparisons
- eval calls
- synchronous blocking loops on large data
- global namespace pollution

- (none)

## VERIFY_CHECKS
- syntax_js
- no_var_usage
- no_eval_usage
