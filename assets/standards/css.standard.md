# CSS STANDARD

## USE
- class selectors over id selectors for styling
- rem and em units for scalable typography
- flexbox or grid for layout
- custom properties (variables) for theme colors
- mobile-first media queries

## DO_NOT
- important overrides
- inline styles in markup
- fixed pixel widths on containers
- vendor prefixes without fallback
- deeply nested selectors beyond three levels

## REQUIRED_FILES
- styles.css

## VERIFY_CHECKS
- no_important_overuse
- has_responsive_rules
