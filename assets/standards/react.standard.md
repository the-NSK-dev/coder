# REACT STANDARD

## USE
- functional components with hooks
- key prop on every list item
- PascalCase for component names
- useEffect dependency arrays
- prop destructuring in function signatures

## DO_NOT
- direct DOM manipulation
- array index as key in dynamic lists
- inline arrow functions in render-heavy JSX
- setState inside render
- mutating state directly

## REQUIRED_FILES
- package.json
- src/App.tsx

## VERIFY_CHECKS
- jsx_valid
- imports_resolve
- no_index_as_key
