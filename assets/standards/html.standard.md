# HTML STANDARD

## USE
- semantic structural tags (header, nav, main, section, article, footer)
- alt attributes on every img element
- meta viewport tag for responsive scaling
- lang attribute on the html element
- label elements associated with every form input

## DO_NOT
- inline style attributes
- deprecated tags (center, font, marquee, big)
- missing doctype declaration
- duplicate id attributes
- empty href or src placeholders

## REQUIRED_FILES
- index.html

## VERIFY_CHECKS
- doctype_present
- title_tag_exists
- no_broken_tags
